# 컴퓨터 어드벤처 🚀

초등학생을 위한 컴퓨터 학습 게임 사이트. 1단원: 컴퓨터의 기초.

## 실행

`index.html`을 더블클릭하면 브라우저에서 바로 열립니다.

## 화면 흐름

```
로그인 (아이디 입력)
   ↓
캐릭터 선택 (20종) + 이름 짓기
   ↓
홈 (프로필 + 단원 카드 목록)
   ↓
1단원 게임: 사격
   ↓
결과 + 레벨업
   ↓
다시 홈
```

## 게임 규칙 — 1단원 사격

- 화면 상단에 제시어 (예: "컴퓨터의 눈")
- 단어가 위에서 떨어짐 (모니터, 키보드, 책상 등)
- 정답을 클릭하면 +100점 (연속 정답 = 콤보 보너스)
- 오답 클릭은 -30점
- 10라운드 진행

## 폴더 구조

```

├── index.html              # 진입점
├── css/style.css           # 디자인 시스템
├── js/
│   ├── data.js             # 캐릭터·단원·라운드 데이터
│   ├── state.js            # localStorage 상태 관리
│   ├── ui.js               # 라우팅 + 헬퍼
│   ├── app.js              # 부트
│   └── screens/
│       ├── login.js
│       ├── select-character.js
│       ├── home.js
│       ├── game1.js
│       └── results.js
└── assets/                 # (향후: 캐릭터 이미지, 사운드)
```

## 진행 상태 리셋

브라우저 콘솔(F12)에서:
```js
localStorage.removeItem("computer_adventure_state_v1");
location.reload();
```

또는 홈 화면 하단의 **"다른 친구로 시작"** 버튼.
